## üìÇ Repository Structure

The labs are organized into two main sections based on the course timeline:

### üìç [Lab Mid](Lab%20Mid/)
Focused on basic networking foundations and initial configuration.
- **Lab 1:** LAN & Basic Router Configuration (`.pkt`)
- **Lab 2:** Router-to-Router Communication
- **Lab 3:** Static & Dynamic Routing Basics
- **Lab Evaluation:** Mid-term practical evaluation files

### üìç [Lab Final](Lab%20Final/)

- **Lab 1:** Advanced Static & Dynamic Routing

- **Lab 3:** Virtual LAN (VLAN) & Inter-VLAN Routing
- **Lab 4:** RIP V2 Protocol Implementation
- **Lab 5:** EIGRP Protocol Configuration with Servers

## üöÄ How to Use These Files
1. **Prerequisite:** Ensure you have [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) installed on your system.
2. **Clone the Repo:**
   ```bash
   git clone https://github.com/MIHMahmudEli/Computer-Networks-Lab.git
   ```
3. **Open Files:** Navigate to the desired folder and double-click the `.pkt` files to open them in Packet Tracer.
4. **Explore:** Analyze the topology, check the CLI configurations, and test connectivity using the simulation/ping tool.
